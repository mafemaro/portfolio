> Portfólio pessoal construído do zero utilizando HTML, CSS e JavaScript (Vanilla). O objetivo deste projeto é apresentar minha trajetória acadêmica, experiência profissional e principais projetos na área de tecnologia.

## 🌟 Funcionalidades

Este portfólio foi desenhado com foco em usabilidade, performance e design moderno, contando com as seguintes interações:

* **Dark Mode (Modo Escuro):** Alternância fluida entre temas claro e escuro, salvando a preferência do usuário no `localStorage` do navegador.
* **Design Responsivo:** Layout fluido que se adapta perfeitamente a dispositivos móveis, tablets e desktops.
* **Animações de Scroll:** Elementos da página surgem suavemente (efeito *fade-in* e *slide-up*) à medida que o usuário rola a tela, utilizando a API `IntersectionObserver`.
* **Efeito de Digitação (Typing Effect):** Animação CSS pura no cabeçalho inicial para simular a digitação de texto.
* **Navegação Suave (Smooth Scroll):** Rolagem suave ancorada nas seções da página.
* **Cards Interativos:** Efeitos de flutuação e brilho (glow) nos cartões de projetos e tags de habilidades.

## 🛠️ Tecnologias Utilizadas

Este projeto não utiliza frameworks externos, focando no domínio das tecnologias base da web:

* **HTML:** Estrutura semântica e acessível.
* **CSS:** Estilização avançada com Flexbox, CSS Grid, Variáveis (Custom Properties) para o Dark Mode e Keyframes para animações.
* **JavaScript:** Lógica para manipulação do DOM, controle de temas e observação de scroll.
* **FontAwesome:** Biblioteca de ícones.
* **Google Fonts:** Tipografia (Poppins).
